Snowdrift.coop Style Guide

Cohesive design on all pages of a site is an important part of good user experience. The goal of this style guide is to make it as easy as possible to implement these styles site-wide.

Typography

Headings

Site headings and subheadings are displayed in a strong slab serif typeface, Patua One.

Body Copy

Longer pieces of copy are displayed in a contemporary sans serif typeface, Raleway.

Color Palette

An analagous color palette has been implemented to give the user a calm and inviting feeling.
